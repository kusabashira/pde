pde
===

Execute processing program quickly.

```
# execute sketch.pde
pde sketch.pde
```

Requirements
-----------

- bash
- processing-java

Usage
-----

```
$ pde [OPTION]... SRC

Options:
  -h, --help      display this help text and exit
  -v, --version   display version information and exit


# execute sketch.pde
pde sketch.pde

# execute ./exercise/first.pde
pde ./exercise/first.pde
```
